# Paneller

A simple ruby api and dsl for setting up panels i.e. to be used with dzen2.

## Installation

Add this line to your application's Gemfile:

    gem 'paneller'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install paneller
